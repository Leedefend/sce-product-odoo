# 列表型页面收敛说明 v1

覆盖页面：`task.center`、`risk.center`、`cost.project_boq`。

## 1. 三页统一项

1. 统一顶部信息层：
   - 页面标题
   - 页面模式标签
   - 记录数量
   - 状态标签
2. 统一工具条：搜索 / 筛选 / 排序
3. 统一批量操作条：选中数量、批量动作、导出
4. 统一状态列样式：状态字段中文化 + 语义色标签
5. 统一金额与比例表达：`万/亿` 与 `%`

## 2. 三页保留差异项

- `task.center`：更强调任务处理节奏（在办/逾期/完成）。
- `risk.center`：更强调风险严重度与预警信号。
- `cost.project_boq`：更强调工程量与金额字段可读性。

## 3. 本轮实现策略

- 不改底层列表 contract 主结构；
- 以 `ListPage` 作为公共骨架承载点；
- 通过 `ActionView` 提供 scene-aware 的 page mode 与记录上下文；
- 保持搜索/筛选/排序能力不破坏。
