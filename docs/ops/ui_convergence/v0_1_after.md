# UI Convergence v0.1 After

## L1/L2/L3 Changes

### Workbench
- L1 kept to 3 blocks: 核心指标 / 今日待办 / 关键风险。
- L2: 项目经营概览、系统建议关注事项（折叠）。
- Header converged: lower-noise chips, weaker diagnostic/meta text, softer background.

### My Work
- Default section keeps user on `待我处理` path.
- Advanced filters are collapsed by default.
- List focuses on action-first columns and row-click direct processing.
- Empty state uses positive statement + two next-step entries.

- Added L1 summary strip: 项目状态与阶段 / 风险摘要 / 关键资金与产值指标。
- Added next-step action rail (risk/todo/contract/cost entry links).
- Detail content remains contract-driven and unchanged in behavior.

### Risk/Contract/Cost (Action surfaces)
- Added surface intent strip with domain-specific purpose and direct entry actions.
- Added action-first quick entries and empty-state next steps.
- Risk/cost/contract/project intents now visible at top for 3-second scan.
